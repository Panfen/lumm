## Lumm
### 1. 介绍
撸猫猫，一款微信小程序，现已关联"IT猫"微信公众号，关注"IT猫"微信公众号可手机真实体验。'撸猫猫'主要功能包括展示猫猫图片，介绍猫猫相关知识，发布猫猫消息等。
